PhD Research Aug 2023 – Present RWTH Aachen University · Aachen, Germany Full-time · 30 hrs/week
Doctoral Researcher — Machine Learning & Inverse Problems
Thesis Preprint
Sparse Exit Wave Reconstruction via Deep Unfolding

Transmission Electron Microscopy (TEM) enables high-resolution imaging of materials. The focus is on exit wave reconstruction — recovery of the complex-valued electron wave at the specimen's exit plane from intensity-only measurements — an inverse problem with a nonlinear forward model. We propose a discretized regularized variational formulation, solve it via proximal gradient algorithm (PGA) unfolded into a neural network, inspired by LISTA, enabling improved reconstruction quality and implicit dictionary learning from data.

Phase Retrieval Deep Unfolding Proximal Gradient Algorithm Complex-Valued Optimization Sparse Reconstruction Rademacher Complexity
Project A08 SFB 1481 — Sparsity & Singular Structures GitHub Thesis (in preparation)
A generalization bound for exit wave reconstruction via deep unfolding

We propose a proximal gradient algorithm (PGA) unfolded into a neural network — inspired by LISTA — where each layer corresponds to one PGA step with learnable parameters, applied to exit wave reconstruction as a phase retrieval problem. This enables improved reconstruction quality, interpretability, and implicit dictionary learning from data. We establish generalization error bounds of order O(√L) using Rademacher complexity.
